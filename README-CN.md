# 来电信息
一个获取来电位置和其他信息(如推销、诈骗)的安卓应用。

[![Build Status](https://travis-ci.org/xdtianyu/CallerInfo.svg?branch=master)](https://travis-ci.org/xdtianyu/CallerInfo)

##功能

1\. 查询来电号码并显示悬浮窗，显示位置和其他号码信息。

2\. 主界面显示最近来电列表。

3\. 在主界面可以查询电话号码信息。

4\. 在设置界面可以自定义设置卡片及悬浮窗的颜色。
